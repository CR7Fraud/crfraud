# CRFraud — A Critical Analysis of Cristiano Ronaldo's Career

> An editorial website offering evidence-based journalism on Cristiano Ronaldo's career — grounded in verifiable facts, official statistics, and real sporting context.

**Live website:** [cr7fraud.github.io/crfraud](https://cr7fraud.github.io/crfraud)

---

## About the Project

CRFraud is not a hate site. It is an editorial project that questions the gap between the media narrative built around Ronaldo and what the sporting data actually shows.

Every section is developed with verifiable sources — UEFA, FIFA, fbref.com, Transfermarkt, leading sports press — and written with the sober, analytical tone the subject deserves.

## Editorial Principles

- **Pro-truth, not anti-Ronaldo** — every argument must withstand scrutiny
- **No hyperbole** — analysis like The Athletic, not like a YouTube comment section
- **Verifiable sources** — every claim must be independently confirmable
- **Real sporting context** — statistics without context are dishonest
